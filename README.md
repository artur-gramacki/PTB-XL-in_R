# PTB-XL-in_R
This script is a port of the Python script `example_physionet.py` to R,  available at [https://physionet.org/content/ptb-xl/1.0.3/](https://physionet.org/content/ptb-xl/1.0.3/) 

Additionally, the ability to filter the original signals has been added. Typical filters used include a `5Hz-40Hz` **bandpass** filter and a `50Hz` **notch** filter.
